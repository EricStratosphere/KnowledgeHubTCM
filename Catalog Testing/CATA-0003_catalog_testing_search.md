## **CATA-0003:** Catalog testing - Search  

> **Summary:** Verify that search E-Book feature is working successfully.  <br>

**Preconditions:** _User is authenticated.._  

Scenario 1 

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 |  1 | Launch application.                                 | Verify that the home screen is shown. | 
 |  2 | In the main page, click the hamburger button on the upper left side of the corner.              | Verify that the user is able to expand the sidebar and view the options. |  
 |  3 | In the sidebar, click the `Search` option.              | Verify that the user is redirected to the search-page |
 |  4 | In the search-page, input the name of the desired book in the text-box.              | Verify that the user is able to fetch E-Books that contain the name of the book. | 
   

**Post-conditions:**  

_None_
