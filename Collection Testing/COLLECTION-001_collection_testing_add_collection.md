## **COLLECTION-0001:** Collection testing - Adding new Collection

> **Summary:** Verify that user is able to create collections of E-Books under a specific common quality.  <br>

**Preconditions:** User must be logged in this session and the User's profile page must be accessible.  

Scenario 1 

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 |  1 | Launch application.                                 | Verify that the home screen is shown. | 
 |  2 | In the main page, click the profile icon on the upper right corner.              | Verify that the user is redirected to the user-profile-page. |  
 |  3 | In the book-info-page, tap `Read Now` Button.              | Verify that the user is redirected to the book-doc-page. |  
 |  4 | In the book-doc-page, tap the second icon from the right in the upper right corner of the page.              | Verify that the user is able to see the Modal for annotation. |  
 |  5 | Input desired text for annotation and press `+` in the upper right corner of the modal.              | Verify that the user was successful in adding the notes for that specific page in the book to the database. |  
|  6 | Click the right-most icon in the upper right corner of the page.              | Verify that the user was successful in adding the notes and is visible in the list of appended notes. |  
 
 

**Post-conditions:**  

 - The user will have access to the list of all notes and can choose to traverse to the page they are appended to at any moment.
