## **USER-0004:** User testing - Edit Username

> **Summary:** Verify that username is modified and displayed successfully.  <br>

**Preconditions:** User is authenticated 

Scenario 1 

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 |  1 |   Launch Application   | Verify that the main page is shown   | 
 |  2 |   On the upper right corner of the home page, click on the profile icon on the upper right corner of the page.   | Verify that the user is able to access the user-profile-page.   | 
 |  3 |   In the user-profile-page, below the component that displays the username and e-mail are two buttons. Click the left button that says `Edit Username`   | Verify that the modal to edit the username is shown.  |  
 |  4 |   Input new username in the modal text box and click the check icon in the upper right corner.   | Verify that the new username reflects in the profile page.  |  
**Post-conditions:**  
None.
