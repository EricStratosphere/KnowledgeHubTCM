## **USER-0002:** User testing - Bookmark Page In E-Book

> **Summary:** Verify that user is able to bookmark a page when opening an e-book.  <br>

**Preconditions:** User must be logged in this session and it is verified that the user can successfully open the E-Book.  

Scenario 1 

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 |  1 | Launch application.                                 | Verify that the home screen is shown. | 
 |  2 | In the main page, click a book card among the showcased in the dashboard.              | Verify that the user is redirected to the book-info-page. |  
 |  3 | In the book-info-page, tap `Read Now` Button.              | Verify that the user is redirected to the book-doc-page. |  
 |  4 | In the book-doc-page, tap the open book ico n in the lower left side of the screen next to the slider.              | Verify that the user is able to bookmark the specific page accordingly. |  
 |  5 | In the upper right corner, click the right-most icon to view list of bookmarked pages.              | Verify that the user was successful in adding the bookmarked page to the database. |  
 
 

**Post-conditions:**  

 - The user will have access to the list of all bookmarked pages and can choose to traverse to them at any moment.
